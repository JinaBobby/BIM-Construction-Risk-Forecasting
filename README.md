# AI-Driven BIM-Based Construction Risk & Cost Forecasting System

## Overview
Predicts construction cost overruns and project risk using BIM lifecycle data, time-series forecasting, and machine learning.

## Tools
Jupyter Notebook, Python, Pandas, NumPy, Scikit-learn, TensorFlow/Keras, Statsmodels, Streamlit

## Key Features
- Time-series cost forecasting (ARIMA, LSTM)
- ML-based construction risk prediction
- Feature engineering on BIM data
- Interactive dashboard for decision-makers
